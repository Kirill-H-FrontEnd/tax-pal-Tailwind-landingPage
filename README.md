# TaxPal Landing Page (Next.js)

[![Banner](./public/TaxPalBanner.png)](https://tax-pal-landing-page.vercel.app)

TaxPal is a polished marketing website for a fictional accounting platform. The project demonstrates how to build a modern, responsive landing page with Next.js, TypeScript, and Tailwind CSS, featuring subtle animations, conversion-driven sections, and dark mode support.

## ✨ Features

- Responsive layout designed for marketing content.
- Rich hero, feature, testimonials, and pricing sections tailored to accounting services.
- Animated UI elements powered by Framer Motion for a refined feel.
- Dark mode styling using Tailwind CSS utilities.
- Reusable React components written in TypeScript for type-safe development.

## 🚀 Getting Started

- Clone the repository
- Install dependencies with `npm install`

```bash
 npm install
```

- Run the development server with `npm run dev`

```bash
 npm run dev
```

- Open your browser and navigate to `http://localhost:3000`
